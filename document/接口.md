```java
public interface AAAA{
    
}


public class 类名 implements 接口名{}
public class 类名 implements 接口名1，接口名2{}
public class 类名 extends 父类 implements 接口名1，接口名2{}
```



```
1. 定义类规范, 不是类, 不能实例化对象, 没有构造方法,
2. 所有的方法必须是抽象方法, 实现接口的类必须实现所有方法，否则就必须声明为抽象类
3. 方法只能是 public abstract, 隐式的指定为 public abstract
4. 变量只能是 public static final, 隐式的指定为 public abstract
```
