### 委托表达式

```
用于指定任务的执行人或用于执行任务的类
委托表达式可以是一个类名，一个Spring bean的引用，或者是一个动态表达式




在 spring 中, 注册成组件, 之后表达式调用组件的方法


```





### BPMN 2.0

```
BPMN 2.0（Business Process Model and Notation 2.0）是一种用于描述和建模业务流程的标准化符号和语法规范。它是由 OMG（Object Management Group）组织制定的，旨在提供一种通用的、可视化的业务流程建模语言，使业务分析师、流程设计师和开发人员能够共同理解和交流业务流程
```

```xml
<bpmn2:extensionElements>
	<flowable:properties/>
</bpmn2:extensionElements>
```
